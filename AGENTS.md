## Project Overview

This project is a demo frontend for a Lark Web App.

The current goal is to prove that a React frontend can be opened inside Lark Desktop through the Web App capability.

Current stack:

- React
- TypeScript
- Vite
- Ant Design
- CSS
- Lark Web App configuration
